# Template basado en Primeng 21 y Angular 21

Este proyecto fue creado con Angular CLI 21 y PrimeNG 21.
This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.

## Development server

Para iniciar localmente ejecute los siguientes comandos:

```bash
npm install
```

```bash
ng serve
```

## Building

Para construir el proyecto ejecute:

```bash
ng build
```

Esto compilará tu proyecto y almacenará los artefactos de compilación en el directorio `dist/`. De forma predeterminada, la compilación de producción optimiza el rendimiento y la velocidad de tu aplicación.
